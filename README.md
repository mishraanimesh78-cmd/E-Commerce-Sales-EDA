# E-Commerce-Sales-EDA
This project performs exploratory data analysis on a global e-commerce sales dataset using Python. It involves data cleaning, preprocessing, and visualization to uncover insights such as customer segmentation, regional sales distribution, and product category performance. Libraries like Pandas, NumPy, Matplotlib, and Seaborn are used to analyze trends and present data through meaningful visualizations.

Data Collection:-
Imported the e-commerce sales dataset for analysis.

Data Understanding:-
Explored dataset structure, features, and data types using basic functions.

Data Cleaning:-
Handled missing and duplicate values
Corrected inconsistent data formats
Converted columns to appropriate data types

Data Preprocessing:-
Created new features where required (e.g., profit, sales categories)
Filtered and organized data for analysis

Exploratory Data Analysis (EDA):-
Analyzed sales trends over time
Examined customer segments and regional performance
Identified top-performing products and categories

Data Visualization:-
Used charts (bar plots, line graphs, heatmaps)
Visualized key insights for better interpretation

Insights & Conclusion:-
Summarized key findings
Highlighted patterns, trends, and business insights
